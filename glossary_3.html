Glossary

Term Description
core genes only the genes that are present in EVERY strain
pan genome the entire complement of genes; every gene from every strain
accessory genes the genes that are not core genes
Soft shell genes 95% <= strains < 99%
Shell genes 15% <= strains < 95%
Cloud genes 0% <= strains < 15%
