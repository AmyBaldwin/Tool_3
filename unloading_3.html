Downloading the output files

To transfer the files to your personal computer drag and drop.

Each time Tool 3 runs it makes a new folder inside the output folder, output_tool3.

The new output folder can be found here:

/home/ubuntu/genome_store/Tool_3/output_tool3/output_aaa-bbb-ccc/

The name of the new folder includes the run I.D. (aaa), as well as the identity and core settings (bbb and ccc).
